Il monitoraggio della qualità dell'aria nelle aree urbane rappresenta un elemento di rilievo per la salute pubblica e la gestione ambientale. Questo progetto di tesi presenta AirQualityInsight, un sistema di simulazione per la raccolta, elaborazione e visualizzazione di dati sulla qualità dell'aria in tempo reale, sviluppato usando come caso di studio il comune di Bologna.
Il sistema implementa un'architettura a microservizi che integra sensori simulati distribuiti geograficamente, un broker di messaggistica per la gestione dei flussi di dati, un database non relazionale per la persistenza delle misurazioni ed un'interfaccia web responsive per la fruizione del servizio. I sensori virtuali, posizionati nelle intersezioni stradali principali, generano misurazioni pseudo-realistiche dei principali inquinanti atmosferici (PM2.5, PM10, NO2, O3, SO2) seguendo gli standard dell'Indice Europeo di Qualità dell'Aria (EAQI).
L'applicazione front-end fornisce una dashboard interattiva mobile-first che visualizza i dati attraverso una mappa di Bologna con layer configurabili, heatmap dinamiche per la rappresentazione delle concentrazioni di inquinanti e tabelle statistiche aggiornate in tempo reale. Il sistema implementa inoltre il paradigma Web of Things (WoT) conforme alle specifiche W3C, esponendo ogni sensore come risorsa web accessibile tramite Thing Descriptions standardizzate e API REST.
I risultati dimostrano l'efficacia dell'architettura proposta nella gestione di oltre 400 sensori distribuiti sul territorio, con capacità di elaborare e visualizzare grandi volumi di misurazioni simultanee mantenendo prestazioni soddisfacenti.
AirQualityInsight rappresenta una soluzione scalabile per il monitoraggio ambientale urbano, offrendo uno strumento efficace per la comprensione delle dinamiche di inquinamento atmosferico e supportando processi decisionali basati su dati per la gestione della qualità dell'aria nelle smart cities.
